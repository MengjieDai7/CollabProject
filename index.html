What is Next Generation (NG9-1-1) ?

NG9-1-1 represents the shift from traditional analog 9-1-1 systems to modern IP-based systems. This transformation will improve emergency services by creating a faster, more reliable system, enabling the seamless transmission of voice, data, photos, videos, and text messages from the public to emergency responders.

Why NG9-1-1?

In a mandate from the Canadian Radio-Television and Communications Commission, emergency communications are required now to modernize and leverage current trends in location-based voice, text, photo, video, or sensor-driven messaging.

Where doe GIS fit in?

The mobile device's coordinates will determine the call's routing to the appropriate Public Safety Answering Point (PSAP). These coordinates will be mapped onto the PSAP boundaries. Accurate and current GIS data is essential for this improved system to function optimally and save more lives

Data, data, data!

Prompt, coordinated, and organized collection, storage, and maintenance of reliable GIS data from local government is necessary for NG9-1-1 to ensure precise address location verification, effective call routing, resource allocation, and analytical capabilities.

Breakdown of Project

Three focuses

Alias Names

Alias names are utilized by individuals when calling 9-1-1 instead of staying the formal address of a location they might refer to it as a name E.g. Unionville High School. We need to create a solution that connects the name to the address

Hidden Address

A hidden address is an address that has fallen through the cracks and does not exist in the current database. We need to create a solution to capture these hidden addresses.

Visualization

Create statistical and cartographic visualizations to facilitate assessments of NG9-1-1 address data progress. These visual solutions can display how many address are added by municipality in specific data ranges.
