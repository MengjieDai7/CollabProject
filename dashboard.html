VISUALIZATION OF DATA
Integration Insights

The ArcGIS Online (AGOL) Dashboard and Experience Builder provide a comprehensive and interactive platform for analyzing and visualizing address point data, alias data, and survey responses. By leveraging dynamic filtering, detailed charts, and real-time updates, users gain valuable insights and can make informed decisions. This integration ensures a cohesive and efficient user experience across different data visualization tools.

AGOL Dashboard

The AGOL Dashboard is built on a foundation of an AGOL Web Map containing feature layers with Address Points and survey responses. Using OpenStreetMap as the basemap provides a familiar street-level context, aiding in the analysis and visualization of spatial data. This integration ensures real-time data updates and a clear understanding of the spatial distribution of the data.

AGOL Web Map

The AGOL Dashboard integrates data from an AGOL Web Map , containing feature layers with Address Points and survey responses. By using OpenStreetMap as a basemap, analysts can contextualize the spatial distribution of data, aiding in pattern recognition and comparative analysis.

Dynamic Data Visualization

The Date Selector is crucial for visualizations, connecting to data from Address Points Alias Added and Survey Responses. It drives various visual elements like Pie-Charts for Municipality and Address Type, Serial Charts for Date Range and Present Year comparisons, and several indicators. This selector enhances the dynamic representation of data, ensuring real-time updates and detailed insights.

Configurations

The Municipality Selector and Address Type Selector further refine data visualization by connecting to relevant datasets. These selectors facilitate the creation of various charts and indicators, providing detailed information and enabling users to filter data based on specific criteria. The selectors enhance the user experience by allowing for localized and categorized data exploration.

Enhancements for User Experience

The Time Zone is set to America/Toronto, ensuring accurate time-related data analysis. Users can adjust settings for better visualization, such as resizing elements and expanding areas for improved readability. A reset option allows users to revert to default settings, making the dashboard user-friendly and efficient for data analysis.

AGOL Experience Builder

Experience Builder was used to create three distinct pages displaying address points core data, alias data, and survey data. The header, featuring the York Region logo and a menu for seamless navigation, ensures brand consistency and enhances user experience.

Address Points Data

TThe Address Points page features robust data visualization tools, including diverse widgets that offer multifaceted insights. A SQL-based filter mechanism allows analysts to refine data dynamically by criteria such as municipality and address type, enhancing data analysis.

Survey123 Data

Survey data can be filtered by municipalities using an SQL Expression builder. This page includes line, pie, and bar charts to display survey response trends, types, and review status, with dynamic text updating the number of surveys completed per municipality.

Alias Names Data

The Alias Names page includes a main map widget displaying core address data, flagged and unflagged matches, building footprints, and unmatched alias name points. This visualization helps in identifying and analyzing alias name data effectively.
